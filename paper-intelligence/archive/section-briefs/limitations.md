# Limitations Brief

## Likely Limitations

- Workloads are targeted and may not represent all real agent applications.
- Some result matrices are partial.
- Accuracy and quality preservation need stronger paired analysis.
- StateDrop needs stronger evaluation before becoming a headline result.
- API pricing and prompt caching complicate billed-cost interpretation.
- Related work is not yet verified enough for uniqueness claims.
